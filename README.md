# Website Pinjol Sederhana

Situs pinjaman online demo dengan fitur berikut:

- Pengajuan pinjaman langsung dari formulir
- Simulasi cicilan bulanan dan total pembayaran
- Daftar pengajuan pinjaman
- Tombol pelunasan untuk menyelesaikan pinjaman

## Cara menjalankan

1. Pasang dependensi:

```bash
npm install
```

2. Jalankan server:

```bash
npm start
```

3. Buka browser dan kunjungi:

```text
http://localhost:3000
```

## Struktur proyek

- `server.js`: Backend Express untuk API pinjaman
- `public/`: Berisi frontend HTML, CSS, dan JavaScript
- `data/loans.json`: Penyimpanan sementara untuk data pinjaman

> Aplikasi ini dibuat sebagai demo sederhana. Jangan gunakan data nyata atau produksi tanpa penyesuaian keamanan.
